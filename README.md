# XCP上位机工具

基于PyQt5的串口通信上位机软件，用于XCP协议设备的配置和调试。

## 功能特性

1. **串口配置**
   - 自动检测可用COM口
   - 支持刷新串口列表
   - 波特率选择：1200/2400/4800/9600/19200/57600
   - 默认配置：停止位1、数据位8、无校验

2. **解锁验证**
   - 发送PCPSYF解锁报文
   - 验证解锁状态
   - 状态栏显示解锁状态

3. **EEPROM读写**
   - 读取EEPROM数据
   - 写入EEPROM数据
   - 恢复出厂设置（地址9101，参数42240）
   - 复位MCU（地址150541，参数42240）

4. **参数写入**
   - P/N码写入
   - S/N序列号写入（14位）
   - KVA功率写入

5. **其他功能**
   - 通信日志显示
   - 清空日志
   - 安全退出

## 安装依赖

```bash
pip install -r requirements.txt
```

## 运行程序

```bash
python main.py
```

## 打包EXE

1. 安装打包工具：

```bash
pip install pyinstaller
```

2. 在项目根目录执行：

```bash
build_exe.bat
```

3. 生成文件：

```text
dist/XCPTool.exe
```

## 使用说明

1. 选择正确的COM口和波特率，点击"连接"
2. 点击"解锁"发送解锁命令
3. 点击"验证解锁状态"确认解锁成功
4. 解锁成功后，可进行EEPROM读写和参数写入操作

### 常用操作

- **恢复出厂设置**：点击"恢复出厂"按钮，约10秒完成
- **复位MCU**：点击"复位MCU"按钮
- **读取EEP**：输入地址和长度，点击"读 EEP"
- **写入EEP**：输入地址和数据，点击"写 EEP"
- **写入参数**：填写P/N、S/N(14位)、KVA，点击"发送CTO"

## 注意事项

- 确保设备已正确连接到COM口
- 必须先解锁才能进行读写操作
- 恢复出厂设置需要约10秒时间
- S/N序列号必须为14位
